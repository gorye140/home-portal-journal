# home-portal-journal

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要用于存放各类静态 HTML 页面，这些页面彼此独立，不依赖于特定域名或网站框架。每个页面均可通过 GitHub Pages 等静态托管服务直接访问。

适合用于记录、展示或分享一些轻量级的网页内容。

## 目录结构

```
.
├── index.html          # 默认入口页面
├── pages/              # 存放主要 HTML 页面
│   ├── example1.html
│   └── example2.html
├── assets/             # 静态资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：归档的核心 HTML 页面，可逐步扩充。
- `assets/`：页面引用的样式、脚本或图片等资源，按需组织。

## 页面归档说明

- 每个 HTML 页面均为独立完整的文档，不依赖外部服务或特定后端。
- 页面之间可能共享部分资源（如公共样式），但无强制耦合。
- 归档内容遵循静态网页规范，可离线浏览或部署到任意静态服务器。

## 维护说明

- 欢迎提交 Pull Request 新增或更新页面。
- 新增页面请放入 `pages/` 目录，并确保资源路径正确。
- 若页面使用外部资源（如字体、库），建议使用相对路径或可靠 CDN。
- 请保持页面内容简洁、格式规范，不包含恶意代码或违规内容。
